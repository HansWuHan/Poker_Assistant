## 🎯 游戏操作

### 基本操作
- **F** / **FOLD** - 弃牌
- **C** / **CALL** - 跟注
- **R** / **RAISE** - 加注（会提示输入金额）
- **A** / **ALLIN** - 全下

### 辅助功能
- **Q** / **QUESTION** - 提问模式（Phase 2 将启用AI聊天）
- **H** / **HELP** - 显示帮助
- **S** / **STATUS** - 显示当前状态

### 加注操作
当选择加注时，可以输入：
- 具体数字（如 `100`）
- `min` - 最小加注
- `max` - 全下
- `cancel` / `c` - 取消加注

## 🎲 游戏规则

### 德州扑克基础规则
- **玩家数**: 6人
- **初始筹码**: $1,000
- **小盲/大盲**: $5 / $10
- **游戏回合**: 最多100局

### 游戏流程
1. **翻牌前（Preflop）**: 每人发2张底牌
2. **翻牌（Flop）**: 发3张公共牌
3. **转牌（Turn）**: 发第4张公共牌
4. **河牌（River）**: 发第5张公共牌
5. **摊牌（Showdown）**: 比较牌型，确定赢家

## 🤖 AI对手难度

- **Easy（简单）**: 保守玩法，适合新手练习
- **Medium（中等）**: 平衡策略，接近真实玩家
- **Hard（困难）**: 激进玩法，具有挑战性

当前配置：
- 2个 Easy AI
- 2个 Medium AI
- 1个 Hard AI

## ⚙️ 配置说明

编辑 `.env` 文件可以修改游戏配置：

```bash

GAME_PLAYER_COUNT=6          # 玩家数量

# 调试模式
DEBUG=false
```

## 🐛 常见问题

### Q: 游戏运行时出现 "Operation not permitted" 错误
**A**: 这是文件权限问题，运行以下命令：
```bash
chmod +x main.py
chmod 644 .env
```

### Q: 提示缺少依赖
**A**: 重新安装依赖：
```bash
pip3 install -r requirements.txt
```

### Q: 想要修改AI难度
**A**: 编辑 `poker_assistant/engine/game_controller.py` 中的 `_get_ai_difficulties` 方法

### Q: 游戏卡住或无响应
**A**: 按 Ctrl+C 可以安全退出游戏
